# 재설계 구상
## 목표
하이브리드 로켓 모터의 제원값을 조절해서 /Users/poesy/Documents/Praxis/StaRbuSt_mvp/Flight_simul에서 로켓 무게 14.8kg, 동체직경 120mm, 발사대 길이5m, 발사대 각도 85도인 조건에서 고도 250m 이하, 발사대 탈출속도 15m/s이상(안정적으로 약 17m/s 근처)를 만족하는 추진 성능을 만족하는 모터를 재설계하고자 함.

(참고로 현재 연소시험 결과 기준으로 고도 약 800m 및 탈출속도 약 25m/s 임. 고도가 너무 높음.)

## 재설계 변수
```
%% Tank
u.tank.h = 230; 

unit.tank.h = "mm"; % m, mm, cm, in

u.tank.m = 0.85; % 탱크 내부 산화제 총 질량

unit.tank.m = "kg"; % kg, g, lb, oz
```

```
%% Injector

u.inj.d = 1.4;

unit.inj.d = "mm"; % m, mm, cm, in

u.inj.n = 28; % 인젝터 개수
```

```
%% Fuel

u.fuel.R = 7.5; %주의: 반지름 기준임.

unit.fuel.R = "mm"; % m, mm, cm, in

u.fuel.L = 230;

unit.fuel.L = "mm"; % m, mm, cm, in

u.fuel.N = 7;

```

```
%% Combustion Chamber
굳이 연소실이나 프리 및 포스트 챔버 관련 제원값은 안건드려도 됌. 해당 값들은 연소불안정 분석용이었으나 잘 사용하지 않아. 필요없음.
```

```
%% Nozzle

u.nozzle.Dt = 17;

unit.nozzle.Dt = "mm"; % m, mm, cm, in

u.nozzle.De = 35;

unit.nozzle.De = "mm"; % m, mm, cm, in
```

- 재설계 값 우선순위
	- 1. 인젝터 홀 직경 및 개수
	- 2. 노즐목 직경 및 출구 직경
	- 3. 탱크 높이 및 산화제 질량 (되도록 안건들면 좋으나, 사실상 산화제량을 줄여야 고도가 줄어들기에)
	- 4. 그레인 포트 직경 및 개수, 길이 (되도록 이건 안건들면 좋음. 이미 기존값으로 제작된게 7개 있음.)
