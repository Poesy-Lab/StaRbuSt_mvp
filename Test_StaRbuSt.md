# 소개
- `Test_StaRbuSt.m` 스크립트는 StaRbuSt 시뮬레이션 전체를 실행하고 결과를 처리하는 메인 스크립트입니다.
- 스크립트는 다음 주요 단계를 수행합니다:
    1. MATLAB 환경 초기화 (작업 공간, 명령 창, 그림 창 정리)
    2. 사용자로부터 설정 파일 이름 입력 받기
    3. 필요한 모든 하위 폴더를 MATLAB 경로에 추가
    4. 지정된 설정 파일 (`.mat`)에서 시뮬레이션 입력 구조체 `u`와 단위 구조체 `unit` 로드
    5. `Input.m` 함수를 호출하여 초기 시스템 상태 구조체 `x` 생성
    6. `System.m` 함수를 호출하여 전체 시뮬레이션 실행 및 결과 구조체 `y` 획득
    7. `PlotResults.m` 함수를 호출하여 시뮬레이션 결과 그래프 생성
    8. `GenMatResults.m` 함수를 호출하여 시뮬레이션 결과를 `.mat` 파일로 저장

# 실행 순서

1.  **환경 초기화**:
    - `clc`: 명령 창을 지웁니다.
    - `clear`: 작업 공간의 모든 변수를 삭제합니다.
    - `close all`: 열려 있는 모든 Figure 창을 닫습니다.

2.  **설정 파일 지정**:
    - 사용자에게 `.mat` 확장자를 제외한 설정 파일의 기본 이름을 입력하도록 요청합니다 (`input` 함수 사용).
    - 입력받은 기본 이름에 `.mat` 확장자를 추가하여 `config_file_to_load` 변수에 저장합니다.

3.  **경로 추가**:
    - `addpath(genpath(...))` 명령을 사용하여 다음 폴더 및 하위 폴더 전체를 MATLAB의 검색 경로에 추가합니다:
        - `Input`: 초기화 함수 (`Init_*.m`) 및 관련 마크다운 파일 포함
        - `Props`: 유체 물성치 관련 함수 (`N2O.m`, `CO2.m` 등) 포함
        - `Components`: 각 컴포넌트 모델 함수 (`Tank_*.m`, `Vent_*.m` 등) 포함
        - `System`: 시스템 레벨 함수 (`System.m`, `PreFeed.m`, `LiqFeed.m` 등) 포함
        - `Output`: 결과 플롯 및 저장 관련 함수 (`PlotResults.m`, `GenMatResults.m` 등) 포함
        - `Config`: 설정 파일 (`.mat`) 및 저장 스크립트 (`Save_Input_Config.m`) 포함

4.  **입력 설정 불러오기**:
    - `fullfile('Config', config_file_to_load)`를 사용하여 `Config` 폴더 내의 설정 파일 전체 경로를 구성합니다.
    - `fprintf`를 사용하여 어떤 설정 파일을 불러오는지 사용자에게 알립니다.
    - `exist(config_filepath, 'file')`로 파일 존재 여부를 확인합니다.
    - 파일이 존재하면 `load(config_filepath, 'u', 'unit')` 명령으로 `u` (입력 파라미터 구조체)와 `unit` (단위 정보 구조체) 변수를 작업 공간으로 불러옵니다.
    - 파일이 없으면 오류 메시지를 표시하고 스크립트 실행을 중단합니다.

5.  **초기 상태 생성 (`Input` 함수 호출)**:
    - `[x] = Input(u, unit);` 코드를 통해 `Input.m` 함수를 호출합니다.
    - 이 함수는 로드된 `u`와 `unit`을 바탕으로 모든 컴포넌트의 초기 상태를 계산하고 통합하여 단일 구조체 `x`를 반환합니다.

6.  **시뮬레이션 실행 (`System` 함수 호출)**:
    - `[y] = System(x);` 코드를 통해 메인 시뮬레이션 함수 `System.m`을 호출합니다.
    - `System.m`은 `x` 구조체에 정의된 초기 조건과 설정을 사용하여 시간 단계별 시뮬레이션을 수행하고, 각 시간 스텝의 결과를 저장하여 최종적으로 구조체 `y`를 반환합니다.
    - (주석 처리된 `System_new(x)`와 `PostProcessInterpolateX1(y, x)`는 현재 사용되지 않는 대체 또는 추가 처리 루틴으로 보입니다.)

7.  **결과 후처리 (그래프 생성 및 저장)**:
    - `fprintf`로 후처리 단계 시작을 알립니다.
    - `try-catch` 블록을 사용하여 후처리 중 발생할 수 있는 오류를 처리합니다.
    - **그래프 생성**: `PlotResults(y);`를 호출하여 `Output/PlotResults.m` 스크립트를 실행합니다. 이 스크립트는 시뮬레이션 결과 `y`를 바탕으로 다양한 성능 지표 및 상태 변수의 시간 변화를 보여주는 그래프들을 생성합니다.
    - **결과 파일 저장**: `GenMatResults(y, config_file_base);`를 호출하여 `Output/GenMatResults.m` 스크립트를 실행합니다. 이 스크립트는 시뮬레이션 결과 구조체 `y`를 `.mat` 파일로 저장합니다. 저장될 파일 이름에는 사용자가 초기에 입력한 설정 파일의 기본 이름 (`config_file_base`)이 포함될 수 있습니다.
    - 후처리 중 오류 발생 시 `warning` 메시지를 표시합니다.

# 주요 입/출력

- **주요 입력**:
    - 사용자 입력: `Config` 폴더 내에 존재하는 `.mat` 설정 파일의 (확장자 제외) 이름.
    - 설정 파일 (`.mat`): 시뮬레이션에 필요한 모든 입력 파라미터를 담고 있는 `u` 구조체와 각 파라미터의 단위를 정의하는 `unit` 구조체를 포함해야 합니다.

- **주요 출력**:
    - MATLAB Figure 창: `PlotResults` 함수에 의해 생성된 다수의 시뮬레이션 결과 그래프.
    - `.mat` 파일: `GenMatResults` 함수에 의해 `Output` 폴더 또는 지정된 경로에 저장되는 시뮬레이션 결과 데이터 (구조체 `y` 포함).
    - 명령 창 메시지: 스크립트 실행 과정, 로드되는 파일 정보, 후처리 진행 상황, 오류/경고 메시지 등이 표시됩니다.
