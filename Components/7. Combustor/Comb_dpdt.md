# 소개
- 이 문서는 연소실 내부의 압력 변화율($\frac{dp_c}{dt}$)을 계산하는 **동적(dynamic) 모델**을 설명합니다.
- 이 모델은 연소실 내 질량 및 에너지 보존 법칙에 기반하며, 시간에 따른 압력 변화를 직접 계산합니다.
- 이는 각 시간 스텝에서 유입/유출 유량이 평형을 이룬다고 가정하는 **준정상상태(quasi-steady-state)** 모델 (`Comb_Itercalc.m`이 사용되는 방식)과 대조됩니다.
- 동적 모델은 연소실 내 질량 축적/방출, 연소 온도 변화, (하이브리드 로켓의 경우) 연소실 체적 변화 등에 의한 **과도(transient) 압력 변화**를 모사할 수 있습니다.
- 이 모델을 구현한 함수(가칭 `Comb_dpdt.m`)는 일반적으로 MATLAB의 `ode45`, `ode15s`와 같은 **상미분 방정식(ODE) 또는 미분-대수 방정식(DAE) 솔버**의 입력으로 사용되어 시간에 따른 압력 변화를 적분합니다.
- **주의:** 이 문서는 모델 설명을 위한 것이며, 실제 함수 구현은 추후 진행될 예정입니다.

# Input

동적 압력 변화율 $\frac{dp_c}{dt}$를 계산하기 위해 필요한 입력값들은 특정 시점 $t$에서의 시스템 상태입니다. ODE/DAE 솔버는 일반적으로 시간 $t$와 상태 변수 벡터 $y$를 함수에 전달합니다. $y$ 벡터에는 $p_c$가 포함되며, 다른 필요한 값들은 주로 상태 구조체 `x`를 통해 전달받을 수 있습니다.

| 기호                   | 설명                                                                                                                                  | 예시 변수 (구조체 `x` 내) | 비고                                                                                                             |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------- | 
| $t$                    | 현재 시간                                                                                                                               | - (솔버가 제공)           | ODE/DAE 솔버가 함수 호출 시 제공                                                                                   |
| $p_c(t)$               | 현재 시간 $t$에서의 연소실 압력                                                                                                           | `y(1)` 또는 `x.comb.P`    | 상태 변수 벡터의 일부 또는 `x` 구조체에서 가져옴 (솔버 설정에 따라 다름)                                               |
| $T_c(t)$               | 현재 시간 $t$에서의 연소 온도                                                                                                           | `x.comb.T`              | 현재 $p_c(t)$와 $\text{OF}(t)$에 대한 CEA 계산 결과 (또는 이전 스텝 값 사용)                                     |
| $V_c(t)$               | 현재 시간 $t$에서의 연소실 체적                                                                                                           | `x.fuel.V_port` + ...   | 하이브리드의 경우 포트 부피 등 포함                                                                               |
| $\dot{m}_{ox}(t)$     | 현재 시간 $t$에서의 산화제 유입 유량                                                                                                        | `x.inj.mdot`            | 현재 $p_c(t)$ 등 시스템 상태에 따라 계산됨 (예: `Inj_...` 함수 호출 결과)                                           |
| $\dot{m}_f(t)$        | 현재 시간 $t$에서의 연료 유입 유량                                                                                                          | `x.fuel.mdot`           | 현재 $p_c(t)$, $\dot{m}_{ox}(t)$ 등 시스템 상태에 따라 계산됨 (예: `Grain_...` 함수 호출 결과)                             |
| $\dot{m}_{nozzle}(t)$ | 현재 시간 $t$에서의 노즐 유출 유량                                                                                                        | 계산 필요                 | $p_c(t)$, $T_c(t)$ 등에 따라 계산 (아래 System 참조)                                                             |
| $R$                    | 연소 가스의 기체 상수                                                                                                                  | `x.prop.R`              | CEA 결과 등으로부터 얻음                                                                                          |
| $\dot{T}_c(t)$        | 현재 시간 $t$에서의 연소 온도 변화율                                                                                                      | **계산 어려움**         | 아래 System 및 구현 참고사항 참조                                                                                |
| $\dot{V}_c(t)$        | 현재 시간 $t$에서의 연소실 체적 변화율                                                                                                      | 계산 필요                 | 하이브리드의 경우 연료 후퇴율(`x.fuel.reg_rate`)과 관련됨                                                          |

# System

연소실 압력의 시간 변화율은 다음 미분 방정식으로 표현됩니다:
$$ 
\frac{dp_c}{dt} = \frac{R T_c}{V_c} \left( \dot{m}_{in} - \dot{m}_{out} \right) 
+ p_c\left(\frac{\dot{T}_c}{T_c} 
- \frac{\dot{V}_c}{V_c}\right) 
$$ 
여기서 각 항의 의미는 다음과 같습니다:

1.  **질량 변화 항:** $\frac{R T_c}{V_c} \left( \dot{m}_{in} - \dot{m}_{out} \right)$ 
    *   $\dot{m}_{in} = \dot{m}_{ox} + \dot{m}_f$: 연소실로 유입되는 총 질량 유량입니다. 이 값들은 현재 시스템 상태(압력, 탱크 상태, 연료 상태 등)에 따라 계산됩니다.
    *   $\dot{m}_{out} = \dot{m}_{nozzle}$: 연소실에서 노즐을 통해 빠져나가는 질량 유량입니다. 일반적으로 노즐 목에서의 질식 유량(choked flow)으로 계산됩니다:
        $$ \dot{m}_{nozzle} = \frac{p_c(t) A_t}{\eta_{c^*} c^*(t)} $$ 
        여기서 $c^*(t)$는 현재 $p_c(t)$와 $\text{OF}(t)$에 대한 특성 속도이며, $\eta_{c^*}$는 특성 속도 효율입니다.
    *   이 항은 연소실 내 질량 축적($\dot{m}_{in} > \dot{m}_{out}$) 또는 방출($\dot{m}_{in} < \dot{m}_{out}$)이 압력 변화에 미치는 영향을 나타냅니다. $R T_c / V_c$는 이상기체 상태방정식 $p_c V_c = m R T_c$로부터 유도된 계수입니다.

2.  **온도 변화 항:** $p_c\left(\frac{\dot{T}_c}{T_c}\right)$ 
    *   $\dot{T}_c = \frac{dT_c}{dt}$: 연소 온도의 시간 변화율입니다.
    *   $T_c$는 주로 $p_c$와 $\text{OF}$의 함수($T_c = T_c(p_c, \text{OF})$)이므로, $\dot{T}_c$는 $\dot{p}_c$와 $\dot{\text{OF}}$에 의존하게 되어 계산이 복잡해집니다 (순환 의존성). (자세한 내용은 아래 참고사항 참조)
    *   이 항은 연소 온도가 변할 때 압력 변화에 미치는 영향을 나타냅니다.

3.  **체적 변화 항:** $- p_c\left(\frac{\dot{V}_c}{V_c}\right)$ 
    *   $\dot{V}_c = \frac{dV_c}{dt}$: 연소실 체적의 시간 변화율입니다.
    *   액체 로켓 등에서는 $\dot{V}_c \approx 0$일 수 있지만, **하이브리드 로켓**에서는 연료 그레인이 연소하면서 포트 직경이 증가하므로 $\dot{V}_c > 0$ 이 되어 이 항이 중요할 수 있습니다.
    *   $\dot{V}_c$는 연료의 후퇴율(`reg_rate`)과 현재 포트 형상으로부터 계산될 수 있습니다. (예: 원통형 포트의 경우 $\dot{V}_c = 2 \pi r_p L \cdot \dot{r}_p = 2 \pi r_p L \cdot \text{reg\_rate}$)
    *   이 항은 연소실 체적이 팽창할 때 압력이 감소하는 효과를 나타냅니다.

# Output

이 모델을 구현한 함수(가칭 `Comb_dpdt.m`)의 주된 출력값은 다음과 같습니다:

| 기호          | 설명                     | 예시 변수 | 비고                                      |
| ------------- | ------------------------ | --------- | ----------------------------------------- |
| $\frac{dp_c}{dt}$ | 연소실 압력의 시간 변화율 [Pa/s] | `dpcdt`   | ODE/DAE 솔버가 이 값을 사용하여 $p_c$를 적분 |

# 전체 코드 (예시)

실제 구현 시 ODE/DAE 솔버 (예: `ode45`)의 요구사항에 맞춰 함수 시그니처를 정의해야 합니다. 일반적인 형태는 다음과 같습니다:

```MATLAB
function dydt = Comb_dpdt(t, y, x) 
    % t: 현재 시간 (솔버 제공)
    % y: 상태 변수 벡터 (솔버 제공, 예: y(1) = pc)
    % x: 추가 파라미터 및 상태를 담은 구조체

    %% 1. 현재 상태 변수 추출 및 필요한 값 계산
    pc = y(1); % 현재 연소압
    x.comb.P = pc; % 구조체 업데이트 (다른 계산에 필요할 수 있음)
    
    % 현재 pc, 탱크 상태 등 기반으로 mdot_ox, mdot_f 계산
    % (예: x = Inj_...(x); x = Grain_...(x);)
    mdot_ox = x.inj.mdot;
    mdot_f = x.fuel.mdot;
    mdot_in = mdot_ox + mdot_f;
    
    % 현재 pc, OF 기반으로 Tc, c*, R 계산 (CEA 등 사용)
    % (예: of = mdot_ox / mdot_f; Tc = ..., cstar = ..., R = ...)
    Tc = x.comb.T; % 가정: 이전 스텝에서 Tc 계산/업데이트됨
    cstar = x.comb.cstar; % 가정: 이전 스텝에서 cstar 계산/업데이트됨
    R = x.prop.R; % 가정
    eta_cstar = x.comb.eta;

    % 현재 노즐 유출 유량 계산
    if cstar > 0 && eta_cstar > 0
        mdot_nozzle = (pc * x.nozzle.At) / (eta_cstar * cstar);
    else 
        mdot_nozzle = 0; % 또는 오류 처리
    end
    mdot_out = mdot_nozzle;
    
    % 현재 연소실 체적 Vc 계산
    Vc = x.fuel.V_port + x.other_volumes; % 예시
    
    % Vc의 시간 변화율 dVc_dt 계산 (하이브리드의 경우)
    % (예: reg_rate = x.fuel.reg_rate; dVc_dt = 2*pi*r_p*L*reg_rate;)
    dVc_dt = x.fuel.dVc_dt; % 가정: 이전 스텝에서 계산/업데이트됨
    
    % Tc의 시간 변화율 dTc_dt 계산 (어려움! 아래 참고)
    % --- 접근법 선택 필요 --- 
    dTc_dt_approx = 0; % 예: 가장 간단하게 0으로 가정

    %% 2. dp/dt 계산
    mass_term = (R * Tc / Vc) * (mdot_in - mdot_out);
    temp_term = pc * (dTc_dt_approx / Tc);
    vol_term = - pc * (dVc_dt / Vc);
    
    % Tc 또는 Vc가 0이 되는 경우 방지
    if Tc <= 0 || Vc <= 0
       dpcdt = 0; % 또는 오류 처리
       warning('Comb_dpdt:InvalidState', 'Tc or Vc is non-positive.');
    else
       dpcdt = mass_term + temp_term + vol_term;
    end

    %% 3. 솔버 반환값 설정
    dydt = zeros(size(y));
    dydt(1) = dpcdt; % pc 변화율
    % 만약 다른 상태 변수도 적분 대상이라면 dydt의 다른 요소 설정
    
end 
```

# 구현 참고사항

- **ODE/DAE 솔버:** 이 모델은 `Comb_Itercalc.m`과 `LiqFeed.m`의 `while` 루프 방식 대신, MATLAB의 `ode45`(일반 ODE), `ode15s`(Stiff 문제 또는 DAE) 등의 솔버와 함께 사용되어야 합니다. `System.m` 또는 메인 시뮬레이션 루프에서 ODE 솔버를 호출하고, `Comb_dpdt` 함수를 상태 변화율 계산 함수로 제공해야 합니다.
- **$\dot{T}_c$ 처리 방안:**
    1.  **무시:** 가장 간단한 방법은 $\dot{T}_c \approx 0$으로 가정하는 것입니다. 많은 경우 질량 변화 항이 지배적이므로 합리적일 수 있습니다.
    2.  **근사:** 이전 시간 스텝의 정보를 사용하여 $\dot{T}_c \approx (T_c(t) - T_c(t-\Delta t)) / \Delta t$ 로 근사할 수 있습니다. 현재 $T_c(t)$는 현재 $p_c(t)$로 CEA 계산이 필요합니다.
    3.  **DAE 시스템:** $\dot{T}_c = f(\dot{p}_c, \dot{\text{OF}})$ 관계식을 포함하여 전체 시스템을 DAE로 구성하고 `ode15s`와 같은 DAE 솔버로 푸는 것이 가장 정확하지만 구현이 복잡합니다.
- **다른 상태 변수:** 만약 연료량, 탱크 온도 등 다른 변수들도 시간에 따라 변하는 것을 ODE/DAE 시스템에 포함시킨다면, `Comb_dpdt` 함수는 해당 변수들의 변화율($\dot{m}_{fuel}, \dot{T}_{tank}$ 등)도 계산하여 `dydt` 벡터의 해당 요소에 설정해야 합니다.
